# ロードマップ

## Phase 1 - MVP

最小限の機能でリリースし、ユーザの反応を検証する。

### スコープ

- [x] 要件定義
- [ ] プロジェクト初期構築（Nuxt 3 + Cloudflare）
- [ ] 認証機能（Google OAuth / LINE OAuth / メール+パスワード）
- [ ] 条件入力画面（現在地、予算、時間帯、交通手段）
- [ ] プラン生成機能（Claude API + Google APIs → 2〜3パターン提示）
- [ ] プラン選択・再生成（1セッション最大3回）
- [ ] タイムスケジュール表示
- [ ] 手動リスケ機能（ボタン押下 → 現在時刻・現在地で再提案）
- [ ] 利用回数管理（初回無料、2回目以降 Stripe 課金）
- [ ] 広告表示
- [ ] i18n 基盤導入（日本語のみ）
- [ ] Cloudflare へのデプロイ

### 言語

- 日本語のみ

### リスケ方式

- 手動（ユーザがボタンを押してリスケ発動）
